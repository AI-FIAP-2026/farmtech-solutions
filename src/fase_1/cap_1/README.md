## Descrição da Atividade

Desenvolvimento de uma aplicação Python para gestão agrícola com:

- Suporte a 2 culturas (soja e café)
- Cálculo de área de plantio
- Cálculo de manejo de insumos
- Dados organizados em vetores
- Menu CRUD completo

## Arquivos

- `menu_principal.py` - Sistema principal em Python
- `estatisticas_basicas.r` - Análise estatística em R
- `previsao_do_tempo.r` - Ir Além: API meteorológica em R

## Ir Além
